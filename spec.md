我要一個網站係可以比我用來做劇本殺 月光下的持刀者 的網站

frontend: Vue
backend: python fastapi
database: sqlite

--使用者區分
使用者入去網頁既時候有一個需要輸入pass key
passkey 由後端查詢database
輸入對應密碼（例如001 - 006)即可進入對應角色頁面
輸入 特定passkey e.g. 8888 就可以進入管理員頁面

角色頁面有三個功能
- 網頁需儲存使用者狀態直至其登出

1. 分幕數查看自已角色對應劇本，玩家只能看到目前已開放幕數劇本
2. 共通彷goolge 搜索頁面
    1. 彷goolge 搜索頁面 可以提供玩家搜索 最多3個關鍵字，以空格分開，中其一個就可以顯示多條搜索結果
    2. 玩家只能搜索到目前已開放幕數的線索
3. 共通彷twitter 搜索頁面
    1. 可以搜索對應使用者名稱看到相關內容
    2. 玩家只能搜索到目前已開放幕數的線索
4. 共通彷 WhatsApp 頁面
    1. 可以輸入相關號碼，玩家可以打字詢問問題，中對應字眼有相應既文字回應，要有對話動畫
    2. 玩家只能搜索到目前已開放幕數的線索

管理員頁面有以下功能
- 管理員可以輸入一個數字並更新現時幕數
- 更新成功／失敗顯示相對應提示並顯示目前幕數,
- 可返回passkey頁面
